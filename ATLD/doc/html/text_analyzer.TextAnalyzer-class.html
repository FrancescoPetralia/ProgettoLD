get_all_words_occurrence(self)

source code 

Metodo che colleziona all'interno di un oggetto di tipo FreqDist tutte le parole e le relative occorrenze. Nello specifico, è una lista di tuple avente la seguente struttura: [(parola1, occorrenza), (parola2, occorrenza), ..., ...] :return: ogetto FreqDist delle parole.

get_all_chars_occurrence(self)

source code 

Metodo che colleziona all'interno di un oggetto di tipo FreqDist tutte le frasi e le relative occorrenze. Nello specifico, è una lista di tuple avente la seguente struttura: [(frase1, occorrenza), (frase2, occorrenza), ..., ...] :return: oggetto FreqDist delle frasi.

get_results(self)

source code 

Metodo che raccoglie all'interno di un dizionario tutte le statistiche relative al file analizzato. :return: dizionario contenente tutte le statistiche fatte sul file. Il dizionario ha come chiave il nome dell' analisi effettuata, e come valore il return del rispettivo metodo richiamato.

get_ip_address(self)

source code 

Questo metodo ritorna l'indirizzo ip del NameServer. In questo modo assegno un ip, all'interno della rete, al server. In particolare, tramite un socket UDP (volendo, è possibile specificare anche un trasporto TCP), mi connetto a google.com, in modo tale da farmi assegnare un ip dalla rete. Indirizzo ip che sarà poi assegnato al NameServer. :return: Indirizzo IP del NameServer.
